## 五、开发注意事项

### 5.1 用户端（UniApp）

1. **使用 Wot Design Uni 组件库**
   - 遵循组件库的设计规范
   - 自定义主题色覆盖默认配置

2. **适配方案**
   - 使用 rpx 作为单位
   - 设计稿基准 375px

3. **图表实现**
   - 推荐使用 uCharts 或 ECharts for UniApp
   - 饼图、折线图需要配置对应颜色

### 5.2 管理端（antdv-next-admin）

1. **使用 Ant Design Vue 组件库**
   - 自定义主题变量覆盖默认配置
   - 参考模板布局结构

2. **图表实现**
   - 推荐使用 ECharts 或 @antv/g2
   - 响应式适配

3. **表格功能**
   - 支持排序、筛选、分页
   - 横向滚动适配
