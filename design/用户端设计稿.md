## 七、交互说明

### 7.1 页面切换动画

```
页面切换
├── TabBar 切换
│   └── 无动画，直接切换
├── 进入二级页面
│   └── 从右侧滑入 (300ms)
└── 返回上一页
    └── 向右滑出 (300ms)
```

### 7.2 手势操作

```
手势
├── 首页下拉
│   └── 刷新账目列表
├── 账目列表左滑
│   └── 显示编辑、删除按钮
├── 图表双指捏合
│   └── 缩放时间范围
└── 数字键盘长按删除
    └── 快速清除
```

### 7.3 反馈动效

```
反馈
├── 记账成功
│   └── ✅ 打勾动画 + 轻微震动
├── 保存按钮点击
│   └── 缩放 0.95 → 1.0 (150ms)
├── 分类选中
│   └── 轻微放大 + 边框出现
└── 删除确认
    └── 弹出确认框 + 震动提示
```
