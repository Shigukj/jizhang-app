# 记账App 设计规范

> 版本：v1.0
> 更新日期：2025-03-26
> 设计师：UI设计师

---

## 一、设计理念

### 1.1 设计原则

| 原则 | 说明 |
|------|------|
| **极简高效** | 减少操作步骤，3秒完成记账 |
| **清晰直观** | 数据可视化，一目了然 |
| **温暖友好** | 降低记账压力，增加趣味性 |
| **品质感** | 精致的细节，舒适的使用体验 |

### 1.2 设计风格

- **风格定位**：现代简约、清新温暖
- **视觉语言**：圆角卡片、柔和阴影、渐变点缀
- **情感化设计**：微动效、激励反馈、成就系统

## 三、字体规范

### 3.1 字体家族

**用户端（UniApp 小程序）**
```
主字体：PingFang SC (iOS) / HarmonyOS Sans (华为) / MiSans (小米)
后备字体：system-ui, -apple-system, sans-serif
等宽字体：SF Mono (iOS) / Menlo
```

**管理端（Web）**
```
主字体：-apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, 'Helvetica Neue', Arial
中文字体：'PingFang SC', 'Microsoft YaHei'
等宽字体：'SF Mono', 'Fira Code', Consolas
```

### 3.2 字号规范

```

├── Display      32px / 2rem      大标题、数字展示

├── Body-S       12px / 0.75rem   辅助文字
├── Caption      11px             说明文字
└── Mini         10px             极小提示
```
