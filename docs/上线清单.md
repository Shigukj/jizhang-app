## 三、部署清单

### 3.1 服务器要求

| 项目 | 最低配置 | 推荐配置 |
|------|----------|----------|
| CPU | 1 核 | 2 核 |
| 内存 | 1 GB | 2 GB |
| 存储 | 10 GB | 20 GB |
| 系统 | Ubuntu 20.04 | Ubuntu 22.04 |
| Node.js | v18.x | v20.x |

### 3.2 部署步骤

#### 步骤 1：准备服务器

```bash
# 安装 Node.js
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt-get install -y nodejs

# 安装 pnpm
npm install -g pnpm

# 安装 PM2
npm install -g pm2
```

#### 步骤 2：部署后端

```bash
# 克隆代码
git clone <repo> /opt/jizhang-app
cd /opt/jizhang-app/server

# 安装依赖
pnpm install

# 配置环境变量
cp .env.example .env
# 编辑 .env 设置 JWT_SECRET 等

# 使用 PM2 启动
pm2 start src/index.js --name jizhang-api

# 设置开机自启
pm2 save
pm2 startup
```

#### 步骤 3：部署管理端

```bash
cd /opt/jizhang-app/admin-app

# 安装依赖
pnpm install

# 构建
pnpm build

# 使用 Nginx 托管静态文件
# nginx 配置略
```

#### 步骤 4：部署用户端

```bash
cd /opt/jizhang-app/user-app

# 安装依赖
pnpm install

# 构建 H5
pnpm build:h5

# 或构建微信小程序
pnpm build:mp-weixin
# 使用微信开发者工具上传
```

### 3.3 Nginx 配置示例

```nginx
# 后端 API
server {
    listen 80;
    server_name api.jizhang.example.com;
    
    location / {
        proxy_pass http://127.0.0.1:3000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }
}

# 管理端
server {
    listen 80;
    server_name admin.jizhang.example.com;
    
    root /opt/jizhang-app/admin-app/dist;
    index index.html;
    
    location / {
        try_files $uri $uri/ /index.html;
    }
    
    location /api {
        proxy_pass http://127.0.0.1:3000;
    }
}
```

## 五、回滚方案

### 5.1 后端回滚

```bash
# 查看历史版本
pm2 list

# 回滚到上一版本
pm2 restart jizhang-api --update-env

# 或恢复代码
git checkout <previous-commit>
pm2 restart jizhang-api
```

### 5.2 前端回滚

```bash
# 恢复上一版本代码
git checkout <previous-commit>
pnpm build
# 重新部署静态文件
```

### 5.3 数据库备份

```bash
# 备份 SQLite 数据库
cp /opt/jizhang-app/server/data/jizhang.db /backup/jizhang-$(date +%Y%m%d).db
```
