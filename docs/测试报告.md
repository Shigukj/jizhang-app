## 三、发现的问题与修复

### 3.1 问题：better-sqlite3 原生模块无法编译

**现象：**
```
Error: Could not locate the bindings file
```

**原因：** better-sqlite3 需要编译原生模块，沙箱环境缺少编译工具

**解决方案：** 切换到 sql.js（纯 JavaScript 实现的 SQLite）

**修复文件：** `server/src/config/database.js`

### 3.2 问题：注册时 generateToken 参数错误

**现象：**
```
TypeError: Cannot read properties of undefined (reading 'id')
```

**原因：** 数据库返回对象与预期不符

**解决方案：** 修复 authController.js 中的参数传递

**修复文件：** `server/src/controllers/authController.js`
