## 📱 用户端预览

用户端基于 UniApp 开发，支持以下预览方式：

### 方式1: H5 模式预览
```bash
cd workspace/记账App/user-app
pnpm install
pnpm dev:h5
```
访问地址: `http://localhost:5174` (或控制台显示的端口)

### 方式2: 微信开发者工具
1. 下载并安装 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
2. 打开项目目录: `workspace/记账App/user-app`
3. 选择编译模式为小程序或 H5

### 方式3: HBuilderX
1. 下载并安装 [HBuilderX](https://www.dcloud.io/hbuilderx.html)
2. 导入项目: `workspace/记账App/user-app`
3. 运行 -> 运行到浏览器/小程序模拟器

## 🔌 API 接口测试

### 基础地址
```
http://localhost:3000/api
```

### 常用接口

#### 用户注册
```bash
curl -X POST http://localhost:3000/api/auth/register \
  -H "Content-Type: application/json" \
  -d '{"username":"test","password":"123456","phone":"13800138000"}'
```

#### 用户登录
```bash
curl -X POST http://localhost:3000/api/auth/login \
  -H "Content-Type: application/json" \
  -d '{"username":"admin","password":"admin123"}'
```

#### 获取账单列表 (需登录)
```bash
curl http://localhost:3000/api/bills \
  -H "Authorization: Bearer YOUR_TOKEN"
```

#### 健康检查
```bash
curl http://localhost:3000/api/health
```

## ⚠️ 注意事项

1. **数据库**: 使用 sql.js (内存数据库), 重启后数据会丢失
2. **跨域**: 已配置 CORS, 前端可直接请求 API
3. **认证**: JWT Token 有效期为 7 天
4. **端口冲突**: 如端口被占用, 请修改相关配置文件

## 🚀 快速开始

### 一键启动所有服务
```bash
# 启动后端
cd workspace/记账App/server && pnpm dev &

# 启动管理端
cd workspace/记账App/admin-app && pnpm dev &
```

### 一键停止所有服务
```bash
pkill -f "node.*server"  # 停止后端
pkill -f "vite"          # 停止前端
```
