# Agent Skills

用户端的 AI Agent Skills 集合，用于指导 AI 辅助开发。

## 已安装 Skills

| Skill | 描述 |
|-------|------|
| wot-ui | wot-design-uni 组件库使用指南 |
| uni-page-generator | uni-app 页面生成器 |
| pinia-store-generator | Pinia Store 生成器 |
| global-feedback | 全局反馈组件使用指南 |
| api-request | API 请求规范 |

## 说明

当前项目已集成 wot-starter 的核心 Skills，但**不是完整的 wot-starter 模板**。

Skills 主要用于：
- 统一代码风格
- 规范开发流程
- 提高 AI 辅助开发效率

如需完整迁移到 wot-starter，需要重构项目结构，建议谨慎评估。
